生活压力大，饮食调节抗疲劳

生活压力大，学习工作任务繁重，常常会使人感到力不从心。调节疲劳也有很多种方法，有些人选择运动休息，有的人会用饮食调节。那么，哪些蔬菜抗疲劳最有效？

一、抗疲劳的蔬菜

1、菜花

菜花很多人都喜欢吃，它又叫做花椰菜。菜花不但能够消除疲劳而且还具有抗癌的作用。有研究表明，菜花含有大量的维生素U而且还有多种吲哚类的生物这对于癌症以及消化道疾病有着很大的防治效果。

2、卷心菜

卷心菜也是比较常见的蔬菜，那么卷心菜为何会有抗疲劳的功效呢?卷心菜含有的钾盐特别的多，这种钾盐可以组织体内液体的潴留，有利于消除身体出现的各种疲劳情况。卷心菜不但可以消除疲劳，同时还可以延迟衰老的进程从而延长益寿。


咸蛋黄焗菜花

3、菠菜

我想我们对于菠菜印象最深的就是大力水手吃菠菜变得力大无穷的时候。其实常见的菠菜含有丰富的镁元素，这种元素不但可以加强肌肉的力量，而且还能有效的消除疲劳，对于贫血，肺结核等疾病有着治疗作用。菠菜还含有辅酶Q10,这种物质可以抗衰老。想要消除疲劳，又抗衰的白领MM们可以多吃菠菜。

4、山药

山药自古都是补气，美容长寿的标志。那么你知道山药为什么会消除疲劳吗?其实很简单，山药最大的特点就是有利于我们对于食物的消化吸收，因此抗疲劳就不在话下。

5、芹菜

最新的研究证明，芹菜是促进食欲，润肺止咳的最佳食物。同时芹菜对于慢性疲劳综合症、神经衰弱以及缺铁性贫血等有着极大的辅治作用。


菠菜拌粉丝

6、红肉

许多女性都不喜欢吃红肉，然而，红肉所含的铁是红细胞的基本成分，可以保证向身体的所有器官供氧。缺铁会导致贫血，表现为极度疲乏。铁的最好来源是血肠，肝，红肉类，乳鸽，贻贝等。

7、干果

在进行体力和脑力活动之后，可以嚼一些干果或果干等，能快速补充体力。所以应常在书包里放一些杏干、杏仁或榛子等，以备不时之需。

8、牛奶

每餐最好至少要食用一种乳制品。牛奶可以提供丰富的钙，而钙则是强健骨骼的重要元素。

失眠的时候，喝一杯温热的牛奶：它的色氨酸可以促使形成5—羟基色胺，这种物质可以协助大脑调解睡眠。

当我们出现疲劳感时，你不妨试着让自己放轻松，改善一下自己的饮食方面，多吃一些能给自己充能的美食，这样你会感觉更好。

二、白领怎样消除心理疲劳

1、学会自我调试，及时放松自己

保持心理的平衡和宁静针对精神长期高度紧张，白领应学会自我调试，及时放松自己。如参加各种体育活动；下班后泡泡热水澡，与家人、朋友聊天；双休日出游；还可以利用各种方式宣泄自己压抑的情绪等等。

另外在工作中也可以放松，如边工作边听音乐；与同事聊聊天、谈谈笑话；在办公室里来回走走，伸伸腰；打开窗户，临窗远眺，做做深呼吸等等。同时在复杂紧张的工作中，应保持心理的平衡与宁静，俗话说“心定自然凉”。这就要求白领应养成开朗、乐观、大度等良好的性格，为人处事应该稳健，要有宽容、接纳、超脱的心胸。

2、理安排工作和生活

制定切合实际的追求目标，正确处理人际关系白领之所以精神高度紧张，一方面是由于工作量确实大引起的，另一方面也和白领自身处理问题的态度和方法有关。如众多白领以为只有拼命干，才能得到上司的赏识和加薪、晋升；还有的工作缺乏信心，常常担心自己被抄鱿鱼，或被别人超过等等。

在工作方法上也有问题，如工作不分轻重缓急，事无巨细都亲自干，工作效率低等。对此白领应学会应用统筹方法，以提高工作效率。在工作和生活上，应有明确界限，下班后就应充分休息，而不应还惦记着工作，多参加体力活动，以做到劳逸结合、脑力劳动和体力劳动结合。

如果长期感到力不从心，白领就要重新为自己进行角色定位，重新评估自己的能力和自己的价值目标，如目标过高，就应调整目标，以使自己的目标切合实际。

一些有工作狂倾向的人，应经常问问自己，“是工作为了生活呢，还是生活为了工作”；“是健康和生命重要呢，还是事业重要”；“以健康和生命为成本代价换取事业的发达值不值得”，以使自己意识到问题的严重性，回到正常的生活、工作轨道上来。
